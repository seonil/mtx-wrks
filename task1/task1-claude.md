### 🔴 Quick-Win (6개월 내 성과 가시화) — ★★★

| # | 대상 조직 | 과제명 | 핵심 내용 | 기대효과 |
|:-:|:-:|:--|:--|:--|
| **01** | **연구소** | 사양서·규격서 자동 분석 & 설계 가이드 생성 | OEM SOR/SDS를 LLM이 파싱 → 핵심 요구사항 추출 → 설계 체크리스트 자동 생성 | 사양 검토 리드타임 **70% 단축**, 요구사항 누락 방지 |
| **02** | **품질** | AI 비전 기반 외관·납땜 불량 자동 검출 | 전장 PCB·커넥터를 딥러닝 비전으로 실시간 검사 (기존 AOI 보완) | 불량 유출률 **30-40% 감소** |
| **03** | **영업** | RFQ 자동 분석 & 견적 초안 생성 | RFQ 문서 AI 분석 → BOM 기반 원가 추정 → 견적서 초안 자동 작성 | 견적 작성 시간 **60% 단축** |
| **08** | **품질** | 필드 클레임 패턴 분석 & 근본 원인 추론 | 반품·클레임 데이터를 NLP 군집화 → FMEA 기반 Root Cause 자동 추론 | 클레임 분석 시간 **60% 단축** |
| **09** | **전사** | 사내 지식 통합 검색 & 기술 Q&A 엔진 | 설계 표준·시험 성적서·특허·회의록을 벡터 DB화 → 자연어 검색·답변 | 지식 탐색 시간 **80% 단축** |
